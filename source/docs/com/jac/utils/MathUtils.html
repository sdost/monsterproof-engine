clampUnsignedCompare()method 
public static function clampUnsignedCompare(minVal:Number, maxVal:Number, value:Number):Number

Clamps the value based on value's absolute value, and then the sign is re-applied after the clamping. So if minVal is 5 and maxVal is 10 and value is -13 the returned value is -10; If value was -4, -4 is returned. If value was 6, 6 is returned.

Parameters
minVal:Number — the lowest value can be.
 
maxVal:Number — the highest value can be.
 
value:Number — the value to clamp.

Returns
Number — returns the clamped value with the sign re-applied.
