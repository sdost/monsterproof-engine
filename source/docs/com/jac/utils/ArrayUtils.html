Please note that this only matches against direct types and not inherited types.

Parameters
array:Array — the list to search through.
 
className:String — a string that represents the name of the class to search for.

Returns
Array — returns a list of ArraySearchObjects. If no match is found, an array of 0 length will be returned.

traceArrayAllProps()method 
public static function traceArrayAllProps(array:Array, startIndex:int = 0, endIndex:int = -1):void

Traces out all of the property names and values of each object in the specified array, including the inherited properties.
To trace only direct properties see traceArrayProps()

Parameters
array:Array — the list of objects to trace out.
 
startIndex:int (default = 0) — the first index to start tracing. Defaults to 0.
 
endIndex:int (default = -1) — the last index to trace out. Defaults to -1 which means the last index in the list.

See also

traceArrayProps()
traceArrayProps()method 
public static function traceArrayProps(array:Array, startIndex:int = 0, endIndex:int = -1):void

Traces out all of the direct property names and values of each object in the specified array.
Please note that this traces only the direct properties, not any of the inherited properties. See traceArrayAllProps() to trace out inherited properties as well.

Parameters
array:Array — the list of objects to trace out.
 
startIndex:int (default = 0) — the first index to start tracing. Defaults to 0.
 
endIndex:int (default = -1) — the last index to trace out. Defaults to -1 which means the last index in the list.
