Interfacepublic interface IState extends flash.events.IEventDispatcher
ImplementorsState, StateView, StateViewController

An interface for State compliant objects. Please be sure that the following stateEvents are dispatched at the appropriate times: ENTER_COMPLETE EXIT_COMPLETE ENABLE_COMPLETE DISABLE_COMPLETE
