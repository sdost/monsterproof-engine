public function StateView()

This is a class that can be assigned to an object in the library or created new in code. It fullfills all of the requirements of a State object. This can can be controlled by a StateController, or a StateViewController.

This object can also have sub objects of type StateView. These will automatcially be found and returned through the subViews getter
