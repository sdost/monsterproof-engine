public function StateViewController(stateViewList:Array, container:MovieClip)

This is the controller for each display state of type StateView. This must contain a StateView object that it controls.

Parameters
stateViewList:Array — an array of views to be controlled by this controller.
 
container:MovieClip — a MovieClip that will be used to contain and be the parent of the items in the stateViewList parameter.

enter()method 
public override function enter():void

When called, this will add all of the views and subviews to the display list. Once that has completed, it calls enter() on each of those added views.
